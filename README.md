# 🥊 Cuban Boxing App - Metodología Sagarra

Aplicación PWA (Progressive Web App) para gestión integral de atletas de boxeo cubano, desarrollada para Coach Osvimer Rodríguez.

## 🎯 Características Principales

### 👨‍🏫 Modo Entrenador
- ✅ Gestión completa de atletas (agregar, editar, eliminar)
- ✅ Registro de peso, altura y datos de entrenamiento
- ✅ Sistema de peleas y análisis de rivales
- ✅ Control total del sistema

### 💪 Modo Atleta
- ✅ Acceso a su perfil personalizado
- ✅ Cambio seguro de PIN de acceso
- ✅ Visualización de entrenamientos
- ✅ Seguimiento de peleas

## 🔐 Sistema de Acceso

### Entrenador
- **PIN**: `1967`
- Acceso completo al sistema

### Atletas
- **PIN por defecto**: `1234`
- Cada atleta puede cambiar su PIN desde su perfil
- El nuevo PIN se guarda y funciona de verdad

## 🚀 Cómo Usar

### Instalación Local
1. Descarga el archivo `index.html`
2. Abre en tu navegador (Chrome, Firefox, Safari, Edge)
3. ¡Listo! No requiere servidor

### Instalación como App (Mobile)
1. Abre en navegador móvil
2. Menu ⚙️ → "Instalar app"
3. Se instala como app nativa en tu dispositivo
4. Funciona offline (datos guardados localmente)

### En GitHub Pages
1. Sube `index.html` a tu repositorio
2. Activa GitHub Pages en Settings
3. Accede via `https://tu-usuario.github.io/tu-repo/`

## 💾 Almacenamiento de Datos

Todos los datos se guardan **localmente** en el navegador:
- Atletas
- Peleas
- Preferencias de idioma
- Sesiones activas

**Nota**: Los datos se borran si limpias el cache del navegador. Haz backup regularmente exportando tus datos.

## 🔧 Funcionalidades Principales

### ✏️ Gestión de Atletas
- Crear nuevo atleta (PIN automático: 1234)
- Editar datos (nombre, peso, altura)
- Eliminar atletas con confirmación
- Cada atleta puede cambiar su PIN

### 🔐 Seguridad
- PIN de 4 dígitos para cada atleta

- Los cambios de PIN se guardan de verdad

### 📅 Sistema de Sesiones
- Mantiene la sesión activa
